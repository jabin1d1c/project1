Javascript often abbreviated as JS, is a high-level interpreted programming language. It is a laguage which is also characterized as dynamic, weakly typed, prototype=based and multi-paradigm. Alongside HTML and CSS, Javascript is one of the three core hechnologies of the World Wide Web. Javascript enables interative web pages and thus is an essential part of web applications. The vast majority of websites use it, and all major web browsers have a dedicated Javascript engine to execute it.
